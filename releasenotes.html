Versie 2.3

De complete lijst met issues voor deze release is beschikbaar in de tracker: milestone 2.3 issues.

Nieuw

  • Gebruik Flamingo versie 4.6.4-RC1, zie ook: Flamingo release notes

Verbeteringen en oplossingen

  • Layout factsheet
  • Het filtermechanisme werkt nu consequent
  • De factsheet werkt nu op onze productieomgeving. Was configuratie issue doordat er aliassen waren ingesteld.
  • Editen en splitsen functioneert werkt nu consequent hier op de ontwikkelomgeving.

  • Styling van topmenu; deze mag transparent zijn maar geeft een blauwe balk. In het html component in de header wordt een div toegevoegd met een blauwe achtergrond.
  • Editen van bedrijventerreinen in IE10 (hier ook zonder flow getest) werkt.
